# Iswar Chandra Rana - Portfolio Website

This is a personal portfolio website of Iswar Chandra Rana built with React to showcase my experience, projects, skills, and certifications.

## Features

- Responsive design optimized for all devices
- Interactive UI with smooth scrolling
- Project showcase section
- Skills and experience timeline
- Contact form
- Deployed with GitHub Pages

## Technologies Used

- React.js
- Tailwind CSS
- Framer Motion
- Radix UI Components
- JavaScript
- GitHub Pages for hosting

## Local Development

1. Clone the repository
   ```
   git clone https://github.com/Iswar-Ch-Rana/iswar-ch-rana.github.io.git
   cd iswar-ch-rana.github.io
   ```

2. Install dependencies
   ```
   npm install
   ```

3. Start the development server
   ```
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Deployment

To deploy to GitHub Pages:
```
npm run deploy
```

This will build the project and deploy it to GitHub Pages.
